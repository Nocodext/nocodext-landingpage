# Nocodext — Landing Page Brief

## Positionnement
Nocodext comble les manques de Bubble. On ne remplace pas Bubble, on complète ce qu'ils n'ont pas construit.

## Ton : arrogance tranquille, taquin, assumé
- Pas lisse. Pas complaisant. Pas trop gentil.
- On s'assume : on fait ce que Bubble n'a pas fait.
- Le tacle est taquin, jamais hostile — on respecte Bubble, on surpasse ses limites.
- Mémorable plutôt que corporate.

## Phrase signature
> "The tools Bubble forgot to build."

Ce que cette phrase fait :
- Positionne Nocodext comme le complément naturel de Bubble
- Tacle Bubble sans agressivité — "forgot" est taquin, pas hostile
- Implique : ils n'y ont pas pensé, ou ce n'était pas leur priorité. Nocodext l'a fait.
- Porte implicitement : "ils les ont pas buildés… nous oui."

## Ce qu'on évite
- "power tools" → sonne Microsoft/enterprise
- "supercharge" seul → trop vague, trop vu
- "missing in Bubble" → neutre, plat, ne dit pas pourquoi ni qui apporte la solution
- Formulations corporates, superlatifs creux, promesses floues

## Exemples de copy dans le bon ton
- "Get early access to the tools Bubble forgot to build."
- "Build on Bubble. Go further with Nocodext."
- "Bubble builds the canvas. We build the rest."

## Notes pour Lovable
- Ton : confiant, direct, un peu provocateur mais friendly
- Cible : Bubble builders intermédiaires/avancés qui ont déjà ressenti les limites
- Éviter le design trop sage / trop SaaS générique
- L'arrogance doit transparaître dans le copy, pas seulement dans le design
